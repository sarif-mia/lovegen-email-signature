# Sarif Mia Email Signature

This project contains a professional email signature template for Sarif Mia with SVG icons, improved styling, and responsive design.

## File Structure

```
sarif-mia-email-signature/
├── index.html          # Main HTML file for the email signature
├── css/
│   └── style.css       # Stylesheet for the email signature
├── netlify.toml        # Netlify deployment configuration
└── js/                 # JavaScript files (if needed)
    └── script.js
```

## Features

- Professional design with enhanced typography
- SVG icons for crisp rendering at any resolution
- Modern color scheme with hover effects
- Responsive layout optimized for email clients
- Direct integration with company social media profiles
- Clean separation of HTML, CSS, and presentation

## Usage

1. The signature uses SVG icons directly in the HTML (no image files needed)
2. Update company information in `index.html` as needed
3. Customize styles in `css/style.css` as needed
4. The logo is loaded from the company CDN

## Notes

- This signature is designed to work across major email clients
- SVG icons provide crisp rendering at any resolution
- All styling is contained in external CSS for easy maintenance
- The signature has been customized with  SARIF MIA information
- Ready for deployment to Netlify or other hosting platforms
