# LoveGen Email Signature

A professional email signature template for LoveGen with logo, contact information, and social media links.

## Features

- Responsive design optimized for email clients
- Integrated social media icons
- Easy to customize

## Usage

1. Open `index.html` in your browser
2. Copy the signature HTML
3. Paste into your email client's signature settings

## Files

- `index.html` - Main signature file
- `assets/css/style.css` - Stylesheet
- `assets/social-media-icon/` - Icon images
